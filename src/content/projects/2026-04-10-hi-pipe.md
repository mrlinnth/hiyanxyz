How to Use

Visit the app URL above

Select "Offline / localStorage" option

View deals organized by stage

Click deals to edit, drag between stages to move them

Use filters to focus on specific deals

Screenshots

Main Board View

Main kanban board showing deals organized by stages



Deal Management

Deal card interface showing details and editing options



Features

Visual kanban board for sales pipeline

Drag and drop deal cards between stages

Filter by period, sector, or tags

Real-time totals per stage

Works offline with automatic sync

Installable as web app

Development

git clone https://github.com/mrlinnth/hi-pipe.git

cd hi-pipe

cp .env.example .env

npm install

npm run dev

Tech Stack

React 18 + Vite

Cockpit CMS API

Plain CSS

Docker + Nginx

Environment Variables (optional for offline and local storage usage)

Required in .env:



VITE_COCKPIT_API_URL=https://cms.hiyan.xyz/:hi-pipe/api

VITE_COCKPIT_API_KEY=your_api_key_here
