# Yan's Portfolio

Personal portfolio website built with Astro, Tailwind CSS, and TypeScript.

## Features

- 100/100 Lighthouse performance
- Responsive design
- Light/dark theme
- SEO optimized with auto-generated sitemap and RSS feed
- Type-safe with TypeScript
- MDX support for content

## Tech Stack

- Astro - Static site generator
- Tailwind CSS - Styling
- TypeScript - Type safety
- MDX - Enhanced markdown content

## Commands

| Command | Action |
|---------|--------|
| `npm install` | Install dependencies |
| `npm run dev` | Start local dev server at `localhost:4321` |
| `npm run build` | Build production site to `./dist/` |
| `npm run preview` | Preview build locally |
| `npm run lint` | Run ESLint |
| `npm run lint:fix` | Auto-fix ESLint issues |

## Deployment

The site is deployed via Cloudflare Pages using Wrangler.
